# Reviewer Application Detail — State

## Requirements

### Reviewers can inspect an application before deciding

- Given a reviewer selects an application from their queue, when they open it, then they see the application’s current details, its current workflow state, and the status-change history.
- Given the reviewer cannot access that application, when they try to open it, then the product keeps them out of the detail view.
